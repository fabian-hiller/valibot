# emoji

Creates a pipeline validation action that validates an emoji.

```ts
const Validation = emoji<TInput>(message);
```

## Generics

- `TInput` <Property {...properties.TInput} />

## Parameters

- `message` <Property {...properties.message} />

### Explanation

With `emoji` you can validate the formatting of a string. If the input is not an emoji, you can use `message` to customize the error message.

## Returns

- `Validation` <Property {...properties.Validation} />

## Examples

The following examples show how `emoji` can be used.

### Emoji schema

Schema to validate an emoji.

```ts
const EmojiSchema = string([emoji('Please provide a valid emoji.')]);
```
