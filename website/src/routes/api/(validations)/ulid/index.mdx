# ulid

Creates a pipeline validation action that validates a [ULID](https://github.com/ulid/spec).

```ts
const Validation = ulid<TInput>(message);
```

## Generics

- `TInput` <Property {...properties.TInput} />

## Parameters

- `message` <Property {...properties.message} />

### Explanation

With `ulid` you can validate the formatting of a string. If the input is not an ULID, you can use `message` to customize the error message.

## Returns

- `Validation` <Property {...properties.Validation} />

## Examples

The following examples show how `ulid` can be used.

### ULID schema

Schema to validate an ULID.

```ts
const UlidSchema = string([ulid('The ULID is badly formatted.')]);
```
