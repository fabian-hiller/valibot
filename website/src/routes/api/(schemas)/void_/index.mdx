# void\_

Creates a void schema.

```ts
const Schema = void_(message);
```

## Parameters

- `message` <Property {...properties.message} />

### Explanation

With `void_` you can validate the data type of the input and if it is not `undefined`, you can use `message` to customize the error message.

## Returns

- `Schema` <Property {...properties.Schema} />
