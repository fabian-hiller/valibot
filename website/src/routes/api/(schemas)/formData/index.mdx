# formData

Creates a form data schema.

> The `FormData` class is not available by default in Node.js v18 and below.

```ts
const Schema = v.formData<TMessage>(message);
```

## Generics

- `TMessage` <Property {...properties.TMessage} />

## Parameters

- `message` <Property {...properties.message} />

### Explanation

With `formData` you can validate the data type of the input. If the input is not form data, you can use `message` to customize the error message.

## Returns

- `Schema` <Property {...properties.Schema} />

## Examples

The following examples show how `formData` can be used.

### Form data schema

Schema to validate a form data.

```ts
const FormDataSchema = v.formData();
```
