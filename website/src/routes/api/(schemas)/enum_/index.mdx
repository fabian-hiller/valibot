# enum\_

Creates an enum schema.

```ts
const Schema = enum_<TEnum>(enum_, message);
```

## Generics

- `TEnum` <Property {...properties.TEnum} />

## Parameters

- `enum_` {/* prettier-ignore */}<Property {...properties.enum_} />
- `message` <Property {...properties.message} />

### Explanation

With `enum_` you can validate that the input corresponds to an enum option. If the input is invalid, you can use `message` to customize the error message.

## Returns

- `Schema` <Property {...properties.Schema} />

## Examples

The following examples show how `enum_` can be used.

### Direction enum

Schema to validate a direction enum option.

```ts
enum Direction {
  Left,
  Right,
}

const DirectionSchema = enum_(Direction, 'Invalid direction');
```
