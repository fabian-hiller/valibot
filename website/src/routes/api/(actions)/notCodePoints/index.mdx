# notGraphemes

Creates a not [code points](https://developer.mozilla.org/en-US/docs/Glossary/Code_point) validation action.

```ts
const Action = v.notCodePoints<TInput, TRequirement, TMessage>(
  requirement,
  message
);
```

## Generics

- `TInput` <Property {...properties.TInput} />
- `TRequirement` <Property {...properties.TRequirement} />
- `TMessage` <Property {...properties.TMessage} />

## Parameters

- `requirement` <Property {...properties.requirement} />
- `message` <Property {...properties.message} />

### Explanation

With `notCodePoints` you can validate the number of code points in a string. If the input does not match the `requirement`, you can use `message` to customize the error message.

## Returns

- `Action` <Property {...properties.Action} />

## Examples

The following examples show how `notCodePoints` can be used.

### Not code points schema

Schema to validate a string with more or less than 8 code points.

```ts
const NotGraphemesSchema = v.pipe(
  v.string(),
  v.notGraphemes(8, 'The string must not have 8 code points.')
);
```
