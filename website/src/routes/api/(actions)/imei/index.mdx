# imei

Creates a pipeline validation action that validates an [IMEI](https://en.wikipedia.org/wiki/International_Mobile_Equipment_Identity).

```ts
const Validation = imei<TInput>(message);
```

## Generics

- `TInput` <Property {...properties.TInput} />

## Parameters

- `message` <Property {...properties.message} />

### Explanation

With `imei` you can validate the formatting of a string. If the input is not an imei, you can use `message` to customize the error message.

## Returns

- `Validation` <Property {...properties.Validation} />

## Examples

The following examples show how `imei` can be used.

### Imei schema

Schema to validate an imei.

```ts
const ImeiSchema = string([imei('The imei is badly formatted.')]);
```
