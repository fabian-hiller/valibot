# checkbox

Creates a checkbox transformation action, turning the string `"on"` into `true` and all other values into `false`.

```ts
const Action = v.checkbox();
```

## Returns

- `Action` <Property {...properties.Action} />

## Examples

The following examples show how `checkbox` can be used.

### Checkbox schema

Schema to validate a checkbox string.

```ts
const CheckboxSchema = v.pipe(v.string(), v.checkbox());
```
