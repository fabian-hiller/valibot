# keyof

Creates a picklist schema of object keys.

```ts
const Schema = keyof<TSchema>(schema);
```

## Generics

- `TSchema` <Property {...properties.TSchema} />

## Parameters

- `schema` <Property {...properties.schema} />

## Returns

- `Schema` <Property {...properties.Schema} />

## Examples

The following examples show how `keyof` can be used.

### Object key schema

Schema to validate the keys of an object.

```ts
const ObjectSchema = object({ key1: string(), key2: number() });
const ObjectKeySchema = keyof(ObjectSchema); // 'key1' | 'key2'
```
