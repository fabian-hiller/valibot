# getFallback

Returns the fallback value of the schema.

```ts
const value = getFallback<TSchema>(schema, info);
```

## Generics

- `TSchema` <Property {...properties.TSchema} />

## Parameters

- `schema` <Property {...properties.schema} />
- `info` <Property {...properties.info} />

## Returns

- `value` <Property {...properties.value} />

## Examples

The following examples show how `getFallback` can be used.

### Fallback string schema

Get the fallback value of a string schema.

```ts
const FallbackStringSchema = fallback(string(), "I'm the fallback!");
const fallbackValue = getFallback(FallbackStringSchema); // "I'm the fallback!"
```
