# getDefault

Returns the default value of the schema.

```ts
const value = getDefault<TSchema>(schema);
```

## Generics

- `TSchema` <Property {...properties.TSchema} />

## Parameters

- `schema` <Property {...properties.schema} />

## Returns

- `value` <Property {...properties.value} />

## Examples

The following examples show how `getDefault` can be used.

### Optional string schema

Get the default value of an optional string schema.

```ts
const OptionalStringSchema = optional(string(), "I'm the default!");
const defaultValue = getDefault(OptionalStringSchema); // "I'm the default!"
```
