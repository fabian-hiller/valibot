# NonNullableInput

NonNullableInput inference type.

```ts
// Create nullable sting schema
const NullableStringSchema = nullable(
  transform(string(), (input) => input.length)
);

// Infer non nullable string input type
type NonNullableStringInput = NonNullableInput<typeof NullableStringSchema>; // string
```
