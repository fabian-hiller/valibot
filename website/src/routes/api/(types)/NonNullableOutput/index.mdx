# NonNullableOutput

NonNullableOutput inference type.

```ts
// Create nullable sting schema
const NullableStringSchema = nullable(
  transform(string(), (input) => input.length)
);

// Infer non nullable string output type
type NonNullableStringOutput = NonNullableOutput<typeof NullableStringSchema>; // number
```
