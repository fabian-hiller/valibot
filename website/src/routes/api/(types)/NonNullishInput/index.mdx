# NonNullishInput

Non nullish input inference type.

```ts
// Create nullish sting schema
const NullishStringSchema = nullish(
  transform(string(), (input) => input.length)
);

// Infer non nullish string input type
type NonNullishStringInput = NonNullishInput<typeof NullishStringSchema>; // string
```
