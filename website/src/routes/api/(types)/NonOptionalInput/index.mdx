# NonOptionalInput

Non optional input inference type.

```ts
// Create optional sting schema
const OptionalStringSchema = optional(
  transform(string(), (input) => input.length)
);

// Infer non optional string input type
type NonOptionalStringInput = NonOptionalInput<typeof OptionalStringSchema>; // string
```
